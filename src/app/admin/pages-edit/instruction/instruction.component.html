Приоритет

Сортировка новостей событий и друзей осуществляется по Приоритету. Чем ниже приоритет, тем выше в списке новость или статья. Приоритет может принимать отрицательное значение. Например: Приоритет -9999; Если во время редактирования новость не стала выше по приоритету, то возможно просто надо обновить страницу в редакторе.

Аватар и картики

Все аватарки лучше всего использовать одного размера (На примере 58 X 40). Картинки минимально 250px - максимально 500px. Максимальный размер не более 500 кb. Для уменьшения веса картинки используйте https://tinyjpg.com/. Размер картинки для страницы "друзья" - 375 x 235.

Текст

Для 'приемлемого' отображения текст должен быть не менее чем из 200 символов. При коротком тексте и маленькой картинке может съехать сетка.
